# 🫏 DonkeyDoner 

## ⚙️ Əsas Funksiyalar

* **Routing:** `Flask` çərçivəsi vasitəsilə fərqli səhifələr və keçidlər arasında sürətli naviqasiya təmin edilir.
* **UI:** `templates` qovluğundakı HTML faylları vasitəsilə istifadəçiyə vizual olaraq zəngin və interaktiv bir təcrübə təqdim olunur.
* **Static:** Şəkillər, loqolar və digər vizual elementlər `static/img` qovluğu üzərindən mərkəzləşdirilmiş şəkildə idarə olunur.
* **Backend:** `main.py` faylı tətbiqin əsas mühərriki rolunu oynayır, sorğuları qəbul edir və uyğun cavabları (render_template) geri qaytarır.
* **SQL:** Layihə gələcəkdə verilənlər bazası (SQL) və səbət sistemi əlavə etmək üçün uyğun arxitekturaya malikdir.
